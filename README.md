# Kalorien Meister

Fertige lokale HTML-App.

## Funktionen
- Kalorien manuell eintragen
- Fotoscan-Tool mit Browser-Bilderkennung, Farbanalyse und Komponenten-Schätzung
- sichtbare und mögliche versteckte Bestandteile auswählen
- kcal-Schätzung manuell korrigieren
- Wasser, Gewicht und Schritte eintragen
- jeder Tag lokal gespeichert
- alte Tage über Datum oder Zeitraum öffnen
- Zeitraum-Auswertung und Datenexport

## Wichtig zum Fotoscan
Die App analysiert Bilder direkt im Browser. Wenn Internet vorhanden ist, lädt sie ein Bilderkennungsmodell über CDN. Ohne Internet nutzt sie lokale Pixel-/Farbanalyse. Eine exakte Kalorienmessung ist nur mit Grammangaben/Waage möglich.
